# AI Email Replier

Next.js 16 web app: paste email → pick tone (professional, friendly, brief, apologetic, enthusiastic) → get AI reply → save to history → send via email. Generated by v0. Currently fully client-side and mocked — no AI, no auth, no DB, no email sending.

## Stack
Next.js 16 (App Router) · React 19 · TypeScript strict · Tailwind v4 (tokens in `app/globals.css` via `@theme inline`, no config file) · shadcn on base-ui (not Radix) · next-themes · @vercel/analytics · bun (primary) · lucide-react

RLS: users only access rows where `user_id = auth.jwt() ->> 'sub'`
